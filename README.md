# recipe-app-api
Recipe API project

## Build with Dockerfile
```bash
make d-build
```

## Build with Dockerfile (Linux)
```bash
make d-build-linux
```

## Build with Docker Compose
```bash
make dc-build
```

## Run flake8
```bash
make flake8
```

## Run tests
```bash
make test
```

## Run Project
```bash
make run
```

## Run Commands with Docker
```bash
sudo docker-compose run --rm app sh -c ""
```
